📘 Atividades em JavaScript (Vetores)

Este repositório contém três exercícios de lógica de programação desenvolvidos em JavaScript, utilizando vetores e estruturas básicas como for e while.

O foco é a manipulação manual de arrays, sem uso de métodos prontos.

⚠️ Regras
Não utilizar métodos prontos de arrays (push, map, filter, etc.)
Uso apenas de estruturas básicas (for, while, if)
Entrada de dados com prompt()
Saída de dados com alert()
Conversões com parseInt() e parseFloat()

📂 Exercícios
1. Maior Idade
Lê 4 idades
Armazena em um vetor
Mostra a maior idade e sua posição

2. Funcionárias com Salário Alto
Lê nome, sexo e salário de 5 funcionários
Armazena em vetores separados
Exibe apenas mulheres com salário acima de R$ 5.000

3. Alturas (Quantidade Indeterminada)
Lê nome e altura até o usuário parar (0)
Armazena dados em vetores
Mostra maior e menor altura com seus nomes

📌 Objetivo

Praticar lógica de programação e manipulação de vetores sem uso de funções prontas.
